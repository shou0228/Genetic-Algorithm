# Genetic-Algorithm for APS #自動化生產線排程 #平行機台

✒️排程目標: 

隨著工單逐漸多樣化、產量少且交期短，多機同步作業與換線需求大大增加了排程的難度，使得原本的排程規劃已無法有效提升生產效率，甚至產生閒置，因此我們打算以用 GA 演算法作為 APS (先進規劃排程)的演算邏輯，以 Python 程式語言進行實作來解決此排程難題。
